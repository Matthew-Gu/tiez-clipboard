# TieZ Clipboard Personal Fork

This project is a personal-use fork of [jimuzhe/tiez-clipboard](https://github.com/jimuzhe/tiez-clipboard).

It keeps the local clipboard workflow I use and removes features I do not need, including online synchronization, AI, LAN file transfer, Emoji, theme store, announcements, feedback, and automatic updates.

## Supported Platform

- Windows 10
- Windows 11

## Included Features

- Local clipboard history and search
- Tags, pinning, and deletion
- Quick paste and sequential paste
- Privacy masking
- Built-in themes
- Local data management

## Builds

- The NSIS installer stores data in the normal Windows application data directory.
- The portable ZIP contains `TieZ.exe` and `tiez-data/`. Keep them together so portable data remains beside the executable.
