# TieZ Clipboard 个人 Fork

本项目为 [jimuzhe/tiez-clipboard](https://github.com/jimuzhe/tiez-clipboard) 的个人用途 fork。

项目仅保留本人使用的本地剪贴板工作流，移除了不需要的联网同步、AI、局域网文件传输、Emoji、主题商店、公告、问题反馈和自动更新等功能。

## 支持平台

- Windows 10
- Windows 11

## 保留功能

- 本地剪贴板历史与搜索
- 标签、固定与删除
- 快捷粘贴与顺序粘贴
- 隐私脱敏
- 内置主题
- 本地数据管理

## 构建版本

- NSIS 安装版使用 Windows 正常应用数据目录。
- 便携 ZIP 固定包含 `TieZ.exe` 和 `tiez-data/`，两者保持同级即可将便携数据保存在程序旁。
