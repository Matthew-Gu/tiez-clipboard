## 目标

- 将跨组件 UI 状态迁移到 `useUiStore`。
- 简化 `App.tsx` 编排和组件 props，同时保持列表和快捷键行为不变。

## 执行顺序

- [ ] 迁移搜索、输入法组合状态和类型筛选。
- [ ] 迁移列表选择与键盘模式。
- [ ] 迁移标签编辑和敏感内容显示状态。
- [ ] 迁移设置折叠状态。
- [ ] 使用细粒度 selector，避免订阅完整 Store。
- [ ] 收窄 AppHeader、AppMainContent 和剪贴板渲染器 props。
- [ ] 保留历史、分页、详情缓存和加载状态在现有 hooks。
- [ ] 删除已迁移的 `useAppState` 字段和中间 setter 透传。

## 验收场景

- 搜索、标签搜索、类型筛选和输入法组合行为不变。
- 上下键选择、Enter 粘贴、Escape 行为不变。
- 标签添加、删除、建议和编辑取消行为不变。
- 敏感内容显示与隐藏行为不变。
- 组件只订阅所需状态。
